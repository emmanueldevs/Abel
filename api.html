The Developer API feature.

Using the Developer API

API requests can also be processed by the system, Our developer API is quite simple to use.

To send a request, the user must use the following format where the variables api and url are required. In the example below, the URL of the demo is used but you should use your own domain name. To request a custom alias, simply add &custom= at the end. 

										GET http://localhost/url_shortener/get.php?url=THELONGURLTOBESHORTENED

Server response

As before, the response will encoded in JSON format (default). This is done to facilitate cross-language usage. The first element of the response will always carry the timestamp (timestamp: 23101982). If there is an error, an error message would be displayed, which contains the source of error. otherwise the second element will be named 'url' which contains the short URL. (See below for an example) 

										{
   "timestamp": 23101982,
   "url":"http://localhost/url_shortener/DkZOb"
}
